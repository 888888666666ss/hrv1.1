# 🚀 HR人力资源管理系统 - 部署指南

## 🌐 免费部署选项

### 方案一：Vercel（推荐 - 最简单）

#### 1. 准备工作
```bash
# 确认已构建成功
cd frontend
npm run build  # 已完成，生成了 dist/ 文件夹
```

#### 2. 部署到Vercel
1. 访问 [vercel.com](https://vercel.com)
2. 使用GitHub账号注册/登录
3. 点击 "New Project"
4. 上传 `frontend/dist` 文件夹，或者：

**直接拖拽部署：**
```bash
# 安装Vercel CLI
npm i -g vercel

# 在frontend目录下部署
cd frontend
vercel --prod

# 按提示操作：
# ? Set up and deploy "~/frontend"? [Y/n] y
# ? Which scope? Your username
# ? Link to existing project? [y/N] n
# ? What's your project's name? hr-management-system
# ? In which directory is your code located? ./dist
```

### 方案二：Gitee Pages（国内推荐）

#### 1. 创建Gitee仓库
1. 访问 [gitee.com](https://gitee.com)
2. 注册并创建新仓库
3. 仓库名：`hr-management-system`

#### 2. 上传代码
```bash
cd frontend/dist
git init
git add .
git commit -m "首次部署HR管理系统"
git remote add origin https://gitee.com/你的用户名/hr-management-system.git
git push -u origin master
```

#### 3. 开启Pages服务
1. 进入仓库 → 服务 → Gitee Pages
2. 选择 `master` 分支
3. 点击 "启动" 
4. 获得访问地址：`https://你的用户名.gitee.io/hr-management-system`

### 方案三：腾讯云静态网站托管

#### 1. 开通服务
1. 访问 [腾讯云静态网站托管](https://console.cloud.tencent.com/tcb)
2. 创建环境，选择按量付费（有免费额度）

#### 2. 部署网站
```bash
# 安装腾讯云CLI工具
npm install -g @cloudbase/cli

# 登录
tcb login

# 初始化
tcb init

# 部署
cd frontend
tcb hosting deploy dist -e 你的环境ID
```

## ⚡ 快速部署（推荐Vercel）

### 最简单的方法：

1. **拖拽上传**
   - 打开 [vercel.com/new](https://vercel.com/new)
   - 将 `frontend/dist` 文件夹直接拖拽到页面
   - 等待部署完成（约1-2分钟）
   - 获得免费域名：`xxx.vercel.app`

2. **自定义域名（可选）**
   - 在项目设置中添加自定义域名
   - 支持免费SSL证书
